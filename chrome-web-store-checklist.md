Document why each permission is needed:
- `storage`: User preferences and settings
- `alarms`: Periodic activity checks
- `notifications`: Desktop notifications
- `cookies`: Skool authentication
- `tabs`: Detect Skool pages for content scripts
- `scripting`: Inject UI improvements
- `activeTab`: Current tab access
- `idle`: Optimize background polling
