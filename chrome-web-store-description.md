## 🔧 Technical Details

**Permissions explained:**
- `storage`: Save your preferences and settings
- `alarms`: Schedule periodic checks for new activity
- `notifications`: Show desktop notifications
- `cookies`: Authenticate with your Skool account
- `tabs`: Detect when you're on Skool for content scripts
- `scripting`: Inject UI improvements into Skool pages
- `activeTab`: Access current Skool tab
- `idle`: Optimize polling when you're away

**Host permissions:**
- `skool.com` domains: Required to integrate with Skool
- SKapi API endpoint: Required for automation features
