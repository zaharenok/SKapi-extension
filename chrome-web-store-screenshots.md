# Chrome Web Store Screenshots Guide

## Requirements
- **Size:** 1280x800px (recommended) or 640x400px
- **Format:** PNG or JPG
- **Quantity:** 1-5 screenshots
- **Style:** Clean, professional, showing actual extension UI

## Screenshot Ideas

### 1. Main Dashboard (Primary - 1280x800px)
**Show:** Extension popup with notifications panel and community overview
**Features to highlight:**
- Real-time notifications list
- Member activity feed
- Quick action buttons
- Clean, modern UI

**How to create:**
1. Install extension in Chrome
2. Click extension icon to open popup
3. Take screenshot at 1280x800px resolution
4. Crop/edit to show key features

### 2. Notification Center (1280x800px)
**Show:** Detailed notifications view with alerts
**Features:**

- Integration status

### 4. Community Management (1280x800px)
**Show:** Member management interface
**Features:**
- Member list view

### 5. Free Access Info (1280x800px)
**Show:** Landing page or community join page
**Features:**
- Community join CTA

## Alternative: Use Browser Screenshots

### Quick Method with Chrome DevTools
1. Open Chrome DevTools (F12)
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows)
3. Type "screenshot" 
4. Select "Capture full size screenshot"
5. Edit to 1280x800px using image editor

### Recommended Tools
- **Skitch** - Simple screenshot editing
- **CleanShot X** - Professional screenshots (Mac)
- **Screenshot Guru** - Chrome extension for webpage screenshots
- **Canva** - Template-based screenshot creation

## Design Tips

### DO:
- Use high-resolution images (at least 1280x800px)
- Show actual extension interface
- Keep text readable at small sizes
- Use consistent branding (SKapi.pro colors)
- Highlight key features with visual emphasis
- Include subtle shadows and depth

### DON'T:
- Use low-resolution or blurry images
- Include sensitive user data
- Show error messages or debug info
- Use inconsistent styling across screenshots
- Make text too small to read
- Include browser chrome unless relevant

## Quick Start Checklist

- [ ] Install extension in test Chrome profile

## File Size Guidelines
- Keep each screenshot under 2MB
- Use PNG for UI elements with text
- Use JPG for photographic content
- Optimize compression while maintaining quality
